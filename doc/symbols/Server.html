Class Detail

Server(port, address, debug)
Luo uuden palvelimen annettuun porttiin ja osoitteeseen. Kun palvelimeen tulee dataa, emittoi se siihen kuuluvan eventin paketin ensimmäisen tavun perusteella. esim. NET.LOGIN
Parameters:
{Number} port
Portti, jota palvelin kuuntelee.
{String} address Optional
IP-osoite, jota palvelin kuuntelee. Jos tätä ei anneta, järjestelmä yrittää kuunnella kaikkia osoitteita (0.0.0.0).
{Boolean} debug Optional, Default: false
Spämmitäänkö konsoliin paljon "turhaa" tietoa?
