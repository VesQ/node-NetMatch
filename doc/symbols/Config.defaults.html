Field Detail

<static> {Number} Config.defaults.botCount
Bottien määrä. Jos tämä on nollaa pienempi, käytetään kartan asetuksissa määriteltyä arvoa.
Default Value:
-1
<static> {Number} Config.defaults.botDepartLimit
Minkä pelaajamäärän jälkeen (pelaajat + botit) botteja lähtee itsestään pois. Jos tämä arvo on pienempi kuin botCount-arvo, niin tämä asetetaan samaksi botCountin kanssa. Jos tämä on 0, niin botit eivät poistu automaattisesti koskaan. Jos taas tämä on nollaa pienempi, käytetään kartan asetuksissa määriteltyä arvoa.
Default Value:
-1
<static> {Array} Config.defaults.botWeapons
Bottien käytössä olevat aseet. Jos tämä on tyhjä, käytetään kartan asetuksissa olevia aseita. Jos niitäkään ei ole, on käytössä kaikki aseet. - 1 = Pistooli - 2 = Konepistooli - 3 = Sinko - 4 = Haulikko - 5 = Kranaatinlaukaisin - 6 = Moottorisaha
Default Value:
[]

Default Value:
3000
